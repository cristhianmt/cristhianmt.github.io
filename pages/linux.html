Linea de comandos Linux

  • File system
    1. /: equivalente a C:.
    2. /root: directorio con todos los privilegios del usuario.
    3. /etc: contiene generalmente los archivos de configuracion, cuando y como inician los programas.
      1. /passwd: almacena la información del usuario, se usa para ver todos los usuarios en un sistema.
      2. /shadow : tiene todas las contraseñas de todos los usuarios.
    4. /home: directorio principal del usuario.
    5. /mnt: directorio donde se montan archivos manualmente.
    6. /media: se suelen montar CD y dispositivos extraibles como USB.
    7. /bin: directorio de aplicaciones binarias importantes(el equivalente a ejecutables en Microsoft).
    8. /lib: directorio donde se encuentran todas las librerias del sistema(programas que son similares a los DLL de Microsoft).
    9. /proc: directorio que ofrecer informacion sobre el sistema cerca de los procesos.
    10. /sys: contiene archivos del sistema.
    11. /usr: contiene todos los archivos ejecutables.
    12. sudo -l: imprime los comando permitidos por los usuarios actuales.
    13. Nano /etc/sudoers : dar permisos de ejecución de administrador.
      Debajo de root poner:
      <user> ALL=(ALL:ALL) ALL
  • $: usuario normal.
  • #: super usuario.
  • pwd: directorio de trabajo actual.
  • which: localiza la ruta de los archivos ejecutables.
  • man <app/command>: comando incorporado, da acceso a las páginas del manual para la mayoría de las herramientas.
  • su : permite cambiar de usuario sin cerrar sesión y volver a iniciarla en la sesión primaria.
  • whoami: visualiza que el nombre de usuario actual.
  • ls: listar todos los archivos de directorio actual
    1. -a: muestra todos los archivos/directorios incluidos los “.” ó los ocultos.
    2. -l: muestra los archivos en formato de lista.
  • cd: cambio de directorio.
  • cat <name file>: concatenar archivos lo que proporciona una salida del contenido de cada archivo.
  • >: redireccionamiento para guardar los resultados en un archivo ó sobrescribir el archivo.
  • >>: agrega nueva informacion al archivo creado sin sobrescribir.
  • |: operador tubería donde le permite tomar la salida de un comando y usarla como entrada para un segundo comando.
  • &&: significa "and" permite ejecutar un segundo comado.
  • &: es un operador de segundo plano lo que significa que ejecuta un comando que tarda 10 segundos en ejecutarse, normalmente no podría ejecutar comandos durante ese período; sin embargo, con & ese comando aún se ejecutará y podrá ejecutar otros comandos.
  • ;: es similar a && sin embargo no requiere que el primer comando se ejecute.
  • grep <word>: buscar palabras especificas.
  • chown <user:user> file: nos permite cambiar el usuario y el grupo de cualquier archivo : solo se puede usar si esta ensima de ese otro usuario lo que significa que se realiza mejor con el usuario root
  • find: permite buscar cualquier archivo en el sistema.
    1. /: enumera todos los archivos del sistema operativo.
    2. /<name directory>: enumera todos los archivos del directorio especifico.
    3. -type <type>: enumerar el tipo de archivo.
      1. d: busqueda de directorios.
      2. f: busqueda de archivos.
    4. -name <"*.txt/xml/py">: se especifica el patron a buscar.
      1. -iname: no distingue entre mayúsculas y minúsculas.
      2. ejemplo: find / -type f -name "*.xml"
      3. ejemplo: find /home -type f -iname user.txt
    5. -user <name user>: enumera todos los archivos de un usuario especifico.
    6. -group <name group user >: enumera todos los archivos de un grupo de usuario especifico.
    7. -size <size>: enumera todos los archivos con un tamaño especifico.
      1. n: coincide con el tamaño exactamente.
      2. +n: coincide con valores mayores al tamaño de n.
      3. -n: coincide con valores menores al tamaño de n.
      4. nc: para bytes.
      5. nk: para KiB.
      6. nM: para MiB.
      7. nG: para GiB.
      8. ejemplo: find /home -type f -size -2k -name "*.txt"
    8. -perm <octal(666) ó simbolica(u=r)>: encuentra todos los archivos con permisos especificos.
      1. -: devolvera archivos con al menos esos permisos configurados.
      2. ejemplo: find /usr/bin -type f -user root -perm -u=s --> encontrar todos los archivos de /usr/bin que son propiedad de root y tienen al menos los el permiso SUID
      3. /: devolvera archivos que los se establescan los permisos.
      4. ejemplo: find /usr/bin -type f -user root -perm /666 --> encontrar todos los archivos de /usr/bin que son propiedad de root y que al menos coincidirá con archivos que sean legibles y escribibles por al menos uno de los grupos (propietario, grupo u otros).
    9. time-related: enumera busquedas relacionadas con el tiempo.
      1. prefijos para especificar el tiempo.
        1. a: cuando fue el ultimo acceso.
        2. m: cuando fue la ultima modificacion.
        3. c: cuando fue el ultimo cambio.
      2. valor numerico para especificar el tiempo.
        1. +n: especificar que se accedió por última vez hace mas de n.
        2. -n: especificar que se accedió hace menos de n.
      3. -<prefijo>min: enumera archivos que se especifiquen los minutos.
      4. -<prefijo>time: enumera archivos que se especifiquen los dias.
      5. ejemplo: find /usr/bin -type f -mmin -120 --> encontrar todos los archivos de /usr/bin y que se hayan modificado en las ultimas dos horas.
      6. ejemplo: find /usr/bin -type f -atime +10 --> encontrar todos los archivos de /usr/bin a los que no se accedió en las ultimos diez dias.
      7. 2>/dev/null: suprimir la salida de cualquier error posible para la salida.
      8. ejemplo: find /usr/bin -type f -name "*.txt" 2> /dev/null --> encontrar todos los archivos de /usr/bin con terminacion .txt y ademas que solo aparezca los archivos que se puedan acceder.
  • ps: muestra los procesos que se están ejecutando.
    1. Los números 3-5 digitos que aparece en la segunda columna se le conoce como process ID(PID)
    2. -ef: listar todos los procesos del sistema
  • kill<PID>: mata procesos.
  • chmod <permissions> <file>: permisos de archivo.
    1. Se dividen en tres secciones, usuario y grupo y todos los demás. Todo comienza desde el segundo guión, no desde el primero. Tenga en cuenta que todo está en orden secuencial, por lo que los primeros tres caracteres controlan los permisos del usuario, los segundos tres caracteres controlan los permisos del grupo y los tres últimos caracteres controlan los permisos de todos los demás.
    2. -<permissions>: quitar permisos
    3. +<permissions>: agregar permisos
    1. ejemplo: chmod u-x myfile --> quitar el permiso de ejecusion al usuario.
    2. ejemplo: chmod ug+w myfile --> agregar permisos de escritura al usuario y al grupo.


    1. ejemplo: Chmod 777 <file> --> dar permiso de ejecusion a todos los usuarios.


  • apt <command>: comando en sistemas derivados de debian para instalar programas en Linux.
    1. list : lista los paquetes según los nombres
    2. search: busca en las descripciones de los paquetes
    3. show: muestra detalles del paquete
    4. install: instala paquetes
    5. remove: elimina paquetes
    6. autoremove: Elimina automáticamente todos los paquetes sin utilizar
    7. update: actualiza la lista de paquetes disponibles
    8. upgrade: actualiza el sistema instalando/actualizando paquetes
    9. full-upgrade: actualiza el sistema eliminando/instalando/actualizando paquetes
    10. edit-sources: edita el fichero de información de fuentes
    11. purge <app > : eliminar aplicación
    12. update --fix-missing: error al actualizar los paquetes
  • nano /etc/apt/sources.list: repositorios instalados en el sistema.
  • xprop WM_CLASS: visualiza las propiedades de una ventan.
  • wget: descargardor de archivos no interactivo
  • xrandr -s < display size >: tamaño de pantalla
    1. ejemplo: xrandr -s 1400x1050
  • convert <img> <png, jpg>
  • Shell
    1. echo $SHELL
    2. grep <user> /etc/passwd
    3. nano /etc/passwd
    4. echo $0
    5. getent passwd 'whoami'
  • 7z l: listar que contenido se estara descomprimiendo del comprimido.
    1. x: extraer el archivo.
  • stty -a: informacion sobre la shell.
    1. rows <number>: configurar el numero de filas.
    2. cols <number>: configurar el numero de columnas.
  • dpkg: administrador de paquetes.
    1. -i <packet.deb>: instalacion de paquetes.
    2. -r <name packet>: eliminacion de paquetes columnas.
